# Proyectos Frontend (HTML y CSS)

Esta carpeta contiene mis primeros proyectos desarrollados con HTML y CSS. Están orientados a aprender estructura, estilos básicos y maquetación web desde cero.

## Proyectos incluidos

### **Recetas — Proyecto de Primer Ciclo**
Un proyecto sencillo donde diseñé una página de recetas utilizando:
- estructura HTML básica  
- estilos con CSS  
- imágenes y secciones organizadas  
- navegación simple  

Este proyecto lo realicé como parte de mi formación inicial en desarrollo web y representa mis primeros pasos en el frontend.

## Objetivo de estos trabajos
Explorar lo básico del diseño web, mejorar mi comprensión del HTML y practicar cómo organizar una página con estilos simples.
