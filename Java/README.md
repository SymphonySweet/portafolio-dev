# Laboratorios de Java

Esta carpeta contiene los ejercicios que desarrollé durante mis primeros ciclos del curso de Programación en Java. Los archivos incluyen ejercicios básicos orientados a practicar operadores, condicionales, ciclos, uso de la consola y pequeños programas de lógica.

No son proyectos grandes, pero sí representan mi avance en el lenguaje mientras aprendo a usar NetBeans y a estructurar programas simples.

## Contenido

- Operadores lógicos y aritméticos  
- Condicionales (`if`, `else`, `switch`)  
- Bucles (`for`, `while`, `do while`)  
- Variables y entrada por consola  
- Programas básicos de práctica

Cada archivo pertenece a un laboratorio o práctica distinta realizada durante las clases.
